# HDB Resale Price Analysis (Singapore)

## Project Overview
This project analyzes Singapore HDB resale flat prices to identify trends,
YoY growth patterns, and key factors influencing resale value such as flat age,
town, and flat type.

## Data Source
- Singapore HDB Resale Flat Prices (2017–2024) – Kaggle

## Tools & Technologies
- Python (Pandas, Matplotlib, Seaborn)
- SQL (PostgreSQL)
- Power BI

## Project Structure
- notebooks/ : Data cleaning, EDA, and feature engineering
- sql/ : KPI and YoY growth queries
- docs/ : Insights and analysis documentation
- powerbi/ : Dashboard screenshots

## Key Insights
- HDB resale prices show consistent YoY growth with town-level variation
- Newer flats and longer remaining lease correlate with higher prices
- Price per square foot differs significantly across towns

## Outcome
This project demonstrates an end-to-end analytics workflow from raw data
analysis to business-ready insights and dashboard storytelling.
